Scent – 匂いセンサ 利用ガイド

M5Atom Lite + BME688 を使ったリアルタイム匂いセンサシステムの
セットアップ手順をまとめています。手順どおりに進めてください。

1

ファームウェアの書き込み

M5Atom Lite に専用ファームウェアを書き込みます。
別途インストール不要のブラウザベースの書き込みツールを使います。

  1. M5Atom Lite を PC に USB ケーブルで接続する
  2. 下の「書き込みツールを開く」ボタンをクリックする
  3. ページ内の 接続 ボタンを押し、表示されたシリアルポートを選択する
  4. 書き込み開始 を押して完了まで待つ（初回は 全消去 にチェックを入れることを推奨）
  5. 書き込み完了後、M5Atom Lite が自動で再起動する
対応ブラウザ: Chrome / Edge / Opera（Web Serial API が必要です）
Safari・Firefox には対応していません。

書き込みツールを開く
2

可視化アプリの起動

Windows 11 用の可視化アプリをダウンロードし、ファームウェアを書き込んだ M5Atom Lite と接続してリアルタイムグラフを確認します。

  1. 下の「アプリをダウンロード」ボタンから ZIP ファイルをダウンロードする
  2. ZIP を任意のフォルダに展開する
  3. Scent.exe をダブルクリックして起動する
  4. M5Atom Lite を USB でPCに接続する
  5. アプリ画面の COM ポートドロップダウン から該当ポートを選択し、Connect を押す
  6. グラフが更新され始めたら接続完了
動作環境: Windows 11（64 bit）
注意: セキュリティの警告が表示された場合は「詳細情報」→「実行」を選択してください。
WebView2 Runtime が必要ですが、Windows 11 には標準で搭載されています。
