匂いセンサファームウェア更新ツール

USB接続した M5Atom Lite に、ファームウェアを書き込みます。
対応ブラウザ: Chrome / Edge / Opera (WebSerialに対応している必要があります)

書き込み手順

  1. M5Atom Lite を USB でPCに接続する
  2. 下の 接続 ボタンを押す
  3. 表示されたシリアルポートを選択する
  4. 必要に応じて 全消去 を選択し、書き込み開始を実行する
  5. 完了後に自動再起動を待つ
ブラウザとUSB接続のチェック待ちです。
[ready] ログを表示します。

初回書き込み時は Erase device を有効にするとトラブルを減らせます。
書き込みが失敗する場合は、USBケーブルの交換と、他ツールのシリアルポート占有解除を確認してください。

このページについて

このWebフラッシャーは esptool.js を直接利用しています。
manifest.json とバイナリを読み込み、書き込みフローをアプリ側で制御します。
